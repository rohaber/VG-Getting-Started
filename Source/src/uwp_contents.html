What's included

Once you download the SDK, you can see a folder with several files and sub folders. You can read description of some important files below.

  • Legalthird-party licenses folder
    • VGDRM_SDK_5.2.pdfOpen source used in SDK document
  • VGConnectUWPApplicationVS2015 Solution folder
    • VGConnectUWPApplication.slnVisual Studio 2015 solution
    • VGConnectUWPApplicationDemo application source files
    • libsSDK binaries distribution
      • config.propsSDK configuration file
      • vgdrm_enc.cfgDRMA ecripted configuration file
      • x86Binaries for Intel based platforms
      • armBinaries for Intel based platforms
  • DocsDocumentation folder
    • VGDRM for Universal Windows Platform (UWP) Apps - Developer Guide.pdfUWP Getting started guide
    • VGDRM Application Programming Interface (API) for the Universal Windows Platform (UWP).pdfUWP PI document

Legal Folder

The Legal folder contains third-party licenses.

Note

These licenses should be supplied to the end-user in the installation package.

VGConnectUWPApplication Folder

This folder contains a VS2015 Solution that was used to build the demo application and the binary files required to build your own application using our components.

The VGConnectUWPApplication folder contains the C#/xaml project. This C# project contains references to the set of dlls it expects to find in the folder \Libs\<platform>\, for example, \Libs\ARM\.

GConnectUWPSolution.sln is the VS2015 Solution file. For more information, see section, “Installing and Running Demo Application”.

config.props and vgdrm_enc.cfg are configuration files that must be saved in the folder with binary files. Modify headend properties in file config.props, if necessary.

The appSettings.json file can be used to modify application behavior and sample data.

Note

  • The application reads the localSettings.xml file from the LocalState folder of the Application ( More About Demo Application specifies locations of various LocalState folders).
  • The application can also run without this file.
